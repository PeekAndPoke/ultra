# Kraft FomanticUI Example

Comprehensive showcase of FomanticUI (SemanticUI) components integrated with Kraft.

## What it demonstrates

- Buttons, cards, and menus
- Tables and pagination
- Modals and popups
- Toasts and notifications
- Drag and drop
- Forms and input components

## How to run

```bash
./gradlew -t --parallel :examples:fomanticui:run
```

The development server will start and the app will be available in your browser.
The `-t` flag enables continuous build, so changes are recompiled automatically.
