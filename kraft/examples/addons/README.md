# Kraft Addons Example

Demonstrates all available Kraft addon integrations.

## What it demonstrates

- ChartJS charts and graphs
- Markdown rendering
- Syntax highlighting
- Signature pad (drawing input)
- PDF viewer
- Avatar generation

## How to run

```bash
./gradlew -t --parallel :examples:addons:run
```

The development server will start and the app will be available in your browser.
The `-t` flag enables continuous build, so changes are recompiled automatically.
